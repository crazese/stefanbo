未注册版本和注册版本有什么区别？为什么只能从每组里面导出前两个元素?

未注册版本的限制是：只能分析前两段动作脚本；并且只能导出每个类别的前两项资源；导出的FLA文件不支持动作脚本。
这些限制只针对试用版本，您注册后，可以导出多个文件。

在我决定购买硕思闪客精灵之前，如何使我相信它会毫无问题地将我的swf文件转换成FLA 格式?由于试用版的限制，我不能完全反编译SWF文件中的脚本。

试用版本只支持反编译前两段脚本和每个类别前两项资源的导出，并且导出的FLA文件不支持动作脚本。如果您想在购买前确认文件能够被完全地转换为fla格式，您可以将您的SWF文件发送到我们的Email邮箱：support@sothink.com 我们的技术支持人员将免费的为您测试这个文件，并通过Email告诉您测试结果。然后根据测试结果，您可以决定是否购买这个软件。

为什么我没有在我的IE工具栏中看到硕思闪客名捕？并且当我右键点击Flash影片时， 也没有硕思闪客名捕这个选项?

  1. 请检查您在安装硕思闪客精灵时是否关闭了所有的IE浏览器窗口。 如果没有，请关闭所有IE窗口并重新安装程序。
  2. 如果以前您自定义过您的工具栏，则您的工具栏将不会显示任何一个最近添加的工具栏按钮。请在IE浏览器中点击查看>工具栏>自定义，将SWF捕捉按钮从可用的工具栏按钮中添加到当前工具栏按钮中。这样，硕思闪客名捕的图标将会出现您IE浏览器的工具栏中。
请不要直接在Flash影片上点击右键，这样将显示Flash player的菜单。您应该在Flash影片以外的其他区域点击右键。如果您仍然没有显示硕思闪客名捕，请重新安装硕思闪客精灵。

为什么我不能从网络中获取Flash影片的内容，而只有读取栏或空白影片?

这个问题源于您想捕获的flash采用了loadMovie函数。您可以查看swf文件的动作脚本，看是否有譬如loadMovie之类的脚本，这表明该swf文件在动画中加载了其它的swf文件。这种情况下，硕思闪客精灵不能够直接捕获真正的动画，但您可以使用另外一种方法来手动地捕获。例如，您从硕思闪客精灵中读取该影片的脚本，会发现类似于下面的动作脚本代码：
...
{
loadMovie("dir/file2.swf", ...)
}
...

文件“file2.swf”是真正的动画。然后你可以在浏览器的地址栏中输入地址，如http://www.sothink.com/dir1/file2.swf 来直接打开它。因此你可以使用硕思闪客名捕来从internet上捕获真正的动画。但若你发现LoadMovie中包含了十分复杂不确定的变量，这意味着实际的动画是动态加载的，因此对链接和变量进行解析以获得实际的动画将十分困难。

如何在IE浏览器的弹出窗口中捕获Flash影片?

当您在IE弹出窗口中看见一个Flash影片时，请按如下步骤进行步骤：

  1. 打开一个新的空白IE浏览页；
  2. 将IE图标从弹出窗口的地址栏拖动到新的空白浏览页的地址栏；
  3. 您将会发现Flash影片已在新窗口中打开了；
  4. 现在您可以启动硕思闪客精灵来捕捉它了。

我能够捕捉在Firefox浏览器中弹出的Flash影片吗？如何捕捉?

当然可以，您可以捕捉在Firefox浏览器中弹出的Flash影片，下面是解决方法：

  1. 将鼠标放置在弹出窗口上并点击您的鼠标，选择“About Macromedia Flash Player 8...”，于是您将看到弹出窗口变成了标签模式；
  2. 打开一个新的空白Firefox窗口并将弹出窗口的标签拖动到新窗口；
  3. 您将发现Flash影片已在新窗口中打开了；
  4. 现在您可以使用硕思闪客名捕捕捉Flash影片了。

我能一次导出所有的文件吗?

可以。正式版的硕思闪客精灵支持一次导出所有的文件。将所有要导出的文件夹左边的复选框打上勾，然后单击“导出”即可一次导出所有文件。如果导出swf文件较大，可能导出的时间会稍微长一点，请耐心的等待。

为什么导出的fla文件不能保存在原文件所在的目录中?

在硕思闪客精灵3.x版中，默认情况下，程序将会记录最后一次导出的文件夹，但是您可以依据您的需要来改变存放路径，如果您想把导出的FLA文件存放在原文件所在的目录中，请点击选项>选项，再点击设置按钮，在设置对话框中的一般标签页里，请设置“自动记住最后一次导出的文件夹”选项为未选中状态。

为什么当我在Flash程序中打开导出的FLA文件时会提示“非预期的文件格式”?

硕思闪客精灵3支持导出Flash MX(6.0)、Flash MX 2004(7.0)和Macromedia Flash 8格式的SWF文件。如果您用旧版本的Flash程序中来打开FLA文件，将会出现这个错误，因为通过Flash 5生成的FLA和Flash MX生成的是完全不同的。如果还没有解决您的问题，请发送您的原SWF文件到support@sothink.com.cn来进行检测。

为什么在反编译某些SWF文件后，还生成了除FLA文件外的还有其他一些文件?

如果SWF文件(由Flash 7或更高版本生成)包含动作脚本(Actinscript)2.0 class，在将SWF文件转换为FLA格式后，您将会在某些文件夹中获得除FLA文件外的几个.as文件(所有的.as文件是AS2.0class的源文件)。您也将获得包含所有FLA和.as源文件的工程文件(.flp)。您可以在Flash 7或更高的版本中打开它并使您轻松的管理它们。

为什么硕思闪客精灵不能在导出的FLA文件中保存同原文件一样的元素名称， 并且它们将被转换为什么样的名称?

当您创建一个Flash影片时并生成SWF文件时，层的名称、库元素的名称和元件的名称是不能被保存到SWF文件中的。因此，当使用硕思闪客精灵反编译SWF文件时，这些元素(如层名称、库元素名称和元件名称)将通过硕思闪客精灵重新命名。标签/实例的名称能够同原文件一样，是因为它们被保存在了SWF文件中。

我能用硕思闪客精灵将SWF文件中的文本导出为*.txt格式的文件吗？ 如果可以，应如何导出?

可以，硕思闪客精灵MX2005及更高的版本支持将SWF文件中的文本导出为*.txt格式的文件。在程序中打开SWF文件后，请在资源面板中核对文字元素，然后点击“导出资源”按钮。在资源导出对话框中，您可以看到“文字”的导出格式已被设置为*.txt格式。如果不是，请选择手动Text(*.txt)。这样，您就能够将文字资源导出为*.txt格式的文件了。

怎样导出矢量图?

单击资源栏里矢量图文件夹旁边的+号，所有的矢量图便会展开。把要导出的矢量图左边的复选框打上勾，然后单击“导出”按钮并选择一个文件夹保存这幅矢量图。现在您已经成功地导出了矢量图。硕思闪客精灵支持*.swf和*.gls格式的文件，您可以将矢量图导成这两种格式中的任何一种格式。*.gls格式的矢量图还可以在硕思闪客巫师中使用。

Top

怎样导出图片?

单击资源栏里图片文件夹旁边的+号，所有在SWF文件中的图片都会显示出来。任意点击一副图，这幅图会在预览窗口里看到。把要导出的图片左边的复选框打上勾，然后单击“导出”按钮并选择一个文件夹保存这幅图。现在您已经成功地导出了图片。硕思闪客精灵支持jpg和bmp两种格式的文件，您可以将图片导成这两种格式中的任何一种格式。

Top

怎样导出声音?

单击资源栏里的声音文件夹旁边的+号，所有的声音文件便会展开。点击一个声音文件可以在闪客精灵中试听。把要导出的声音左边的复选框打上勾，然后单击“导出”按钮并选择一个文件夹保存声音文件。现在您已经成功地导出了声音文件。硕思闪客精灵支持.MP3和WAV两种格式的声音文件，您可以将声音导成这两种格式中的任何一种格式。

怎样导出字体/文字/片段动画/按钮/帧？

单击资源栏里字体/文字/片段动画/按钮/帧文件夹旁边的+号，这些元素都会显示出来。将任意一个元素左边的复选框打上勾，然后单击“导出”按钮并选择一个文件来保存这个选中的元素。现在您已经成功地导出了选中的元素。从*.SWF文件中也可以导出字体/文字/片段动画/按钮/帧这些元素。

什么是*.gls 文件，我如何使用它?

当您导出矢量图时，闪客精灵提供您把矢量图存成*.swf或者*.gls格式的文件。*.gls属于硕思闪客巫师自带的一个特殊格式的文件。当您把矢量图导成这种格式的文件，您可以在闪客巫师中使用这副矢量图。具体的使用方法是：先把导出的矢量图在闪客精灵中存成*.gls格式，然后运行闪客巫师程序点击“矢量图”，即转换到矢量图面板。在这个面板中，打开下拉列表，找到刚才您导出的矢量图，现在这个矢量图已经转成了*.gls格式的文件，您可以闪客巫师中使用它。

硕思闪客巫师和硕思闪客精灵是同一个工具吗?

不，这是两个完全不同的工具。硕思闪客巫师是一款创建Flash影片的工具，而硕思闪客精灵是一款用于浏览和分解Flash影片的工具。在闪客精灵中您可以获得很多资源并把这些资源放在闪客巫师中使用。
