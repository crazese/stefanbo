硕思闪客之锤和硕思闪客精灵有什么不同？
硕思闪客之锤是一款Flash影片制作工具，大部分功能和Flash相同。另外，它能导入并且编辑swf文件。硕思闪客精灵是一款用于浏览和解析Flash动画的工具，它只能解析SWF文件并导出成FLA格式，不能用来创作和编辑影片。

闪客之锤有Mac版本吗？
非常抱歉，我们现在只有Windows版本。

为什么在IE工具栏上看不到闪客名捕的图标?
1. 请检查一下您在安装闪客之锤时是否关闭了所有的IE窗口。如果没有，请关闭所有IE窗口然后重新安装程序。
2. 如果您以前自定义过工具栏，工具栏将不会显示新添加的工具按钮。请在IE中点选查看\工具栏\自定义，在打开的对话框中从可选按钮中加入闪客名捕按钮。该图标会自动添加到IE工具栏。

我给实例加了一个特效，但特效重复播放，怎样让特效只播放一遍？
您可以给加入特效的帧加入以下脚本：

1. 在属性面板中给加入特效的实例取一个名字，例如youreffectname.

2. 添加一个新图层，取名为Action，在Action层，点击希望特效开始的那一帧，按下F6加入一个关键帧，在脚本面板中加入以下脚本：

function stopMovieClipRecursively(mc)
{
// stop current effect
mc.stop();
// stop all named movieclips in this movieclip
for (instance_name in mc)
{
if (typeof(mc[instance_name]) == "movieclip")
{
stopMovieClipRecursively(mc[instance_name]);
}
}
}

3. 在 Action层，点击希望特效停止的那一帧，按F6插入一个关键帧并输入以下脚本：

stopMovieClipRecursively(youreffectname);

4. 在特效层，点击希望特效停止的那一帧按下F5插入一帧。

怎样给按钮、影片剪辑或文字添加链接或email链接？
请选中按钮\影片剪辑符号,然后在动作面板中输入以下代码：
on(Release)
{
getURL("在这里填写您要链接的地址", "_blank");
}

Email link:
on(Release)
{
getURL("mailto:在这里填写您的邮箱", "_self");
}

为什么导入的swf文件和sqf文件看上去不同，特别是字体不同？
当您选中影片中的文字，文字面板的字体名变成灰色，这说明您电脑的系统中没有该影片中使用的字体。这种情况下，硕思闪客之锤将以默认字体代替原字体，因此您看到的是不同的字体。

怎样编辑swf文件中的文字？
方法一：请按照以下步骤来改变swf文件中文字的内容：
第一步：在闪客之锤中打开文件，点击“影片浏览器”面板中的“过滤”按钮仅选中“显示文字”的复选框（您可以先选择“全不选”设置为全部不选，再选择“显示文字”）。影片中所有文本元件将显示；
第二步：点击“+”将分类扩展，选择任一个文本元件，该文本元件的实例在画布上被选中；
第三步：选择修改/编辑，现在可以改变文字的内容了；
第四部：关闭对话框，选择文本工具，点击该文本，现在您就可以编辑文字了。

方法二：在库面板中右击文本元件，选择“查看实例...”，在弹出的对话框中，您可以知道元件的具体位置，点击元件名，该文本元件的实例将被激活。

方法三：创建一个文本元件的实例
具体操作如下：从库面板中将文本元件拖到画布上，用文本工具双击文本，文字即处于可编辑状态，您也可以在属性面板中修改文本的相关属性。
请注意：如果给文字加了特效，文本内容将无法编辑。

硕思闪客之锤可以将当前的文件导出成exe或fla格式吗？
非常抱歉，现在的版本还不能支持此功能。您可以在Flash 6或者以上版本中打开SWF文件。然后Flash播放器点击“文件 -->发布设置”将文件转换成exe格式。

硕思闪客之锤能像记事本或者Flash那样将动作脚本复制然后粘贴到别的程序中吗？
可以。现在可以将脚本复制到剪贴板了,您可以通过快捷键Ctrl+c或点击鼠标右键选择复制。

能在SWF文件的动作脚本中搜索到具体某一行脚本吗？还是必须浏览整个脚本才能找到呢？
您可以在“影片浏览器”面板中搜索，点击“过滤”按钮下的小三角形，在下拉列表中仅选中“显示动作脚本”的复选框，然后在旁边的搜索框中输入您要查找的动作脚本，最后点击“搜索”即可找到您需要的动作脚本。

能在拾色器上选颜色吗？
可以，当您打开拾色器时，鼠标指针变成颜色吸管，点击您想要的颜色即可。

怎样在硕思闪客之锤中设置按钮的四种状态？
第一步：进入按钮编辑模式（有Up, Over, Down and Hit, 4种状态)；
第二步：点击按钮的一种状态，在“变形”面板的“颜色转换”中选择“颜色”，设置任何一种颜色，按钮在这个状态时会有不同于其他状态的颜色，当然您也可以为其他状态设置不同颜色。
