CSMate 1.9

  • CSMate 1.9的新特性
    • 完全兼容sXe Injected。请注意：必须先启动CSMate再启动sXe Injected。
    • 支持管理员使用RCON Say命令。
    • 支持管理员使用AMX Say命令。
    • 修正了在某些情况下会显示“修改配置文件失败”，然后退出的问题。

  • 简介
    CSMate是一个免费软件。通过CSMate你可以在玩反恐精英1.6（CS 1.6）和零点行动（CZ）的时候使用中文和其他玩家交谈。


  • 下载
    本站下载：CSMate.zip (296K)

  • 安装运行
    将CSMate.zip解压到任意目录，比如C:\CSMate，然后双击CSMate.exe运行即可。第一次运行时，软件会提示你输入反恐精英的执行文件名（文件名必须是hl.exe，cstrike.exe或者czero.exe）。如果你不记得安装目录，可以右键点击反恐精英的快捷方式，从弹出菜单中选择“属性”，在属性对话框中即可看到对应的目录。

  • 使用
    先运行本软件，然后再启动反恐精英。在游戏时，按“Y”或者“U”会调出输入界面，然后就可以和平常玩游戏一样输入聊天文字。按Enter即可发送，按Esc可取消。点击软件主界面的“选项”按钮可以自定义热键，以及设置弹出/隐藏输入界面时是否自动开启/关闭输入法等。



  • 系统要求
    Windows 2000/XP/2003/Vista
    反恐精英1.6（CS 1.6）或者零点行动（CZ）
