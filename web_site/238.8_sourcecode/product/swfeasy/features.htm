特效

  • 60多种动画效果
    有了闪客巫师软件为您提供的内置特效您可以轻松地创作动画文字、图形和专业的动画。
    给影片加入特效后，您还可以通过改变特效参数来实现不同的动画效果。
  • 创作特效
    闪客巫师支持给一个实例添加入多个特效。点击这里查看叠加特效
  • 保存和运用组合特效
    在闪客巫师软件中有大量组合特效的模板，您可以直接添加到作品中。另外，您也可以自己创作特效，然后放进特效库里，以便下次使用。

沿轨迹运动的特效

内置1000 多个各类素材

在闪客巫师软件中，并不要求您有专业的绘图技术，我们强大的内置资源，包括矢量图文件、图像文件、声音文件，另外还准备了大量的动态按钮和动态背景，在您精心的配置下，一定能创作出专业的动画效果。

在资源库的按钮模板中，每一个按钮上的元素例如：文字、颜色、图形和 URL 等都可以根据需要去修改。把它们用在产品宣传和演示、多媒体课件、电子贺卡中是再好不过了。

在影片剪辑文件夹内为您准备了各式各样的preloader，不用加脚本就可以使用。点击了解如何给影片加preloader?

向导& 模板

  • 操作简易的模板向导
    只需几分钟，跟随向导，即可创作出多媒体相册，广告横幅和导航条。

  • 模板图库
    您可以在模板图库中选择喜欢的内置模板，也可以将自己的作品加到模板图库中以便下次使用。 

按钮

在资源库中的按钮模板，每一个按钮上的元素例如：文字、颜色、图形和 URL 等都可以根据需要去修改。把它们用在产品宣传和演示、多媒体课件、电子贺卡等中是再好不过了。

导入外部SWF的资源

闪客巫师 2005可以导入外部SWF资源，包括图形、图像、按钮、声音和影片剪辑。在Files面板里浏览并选择一个SWF文件，在文件夹前有一个“+”，点击它您就可以找到需要的资源。

绘图工具

  • 熟悉的绘图工具
    闪客巫师软件向您提供：变形工具、改变形状工具、直线工具、铅笔工具、椭圆工具、矩形工具，钢笔工具、文字工具，填充变形工具等熟悉的绘图工具，通过他们您可以创作出不同的元素。
  • 操作简易
    您可以直接在弹出的对话框中设置元素的颜色、填充、位置和大小。
  • 支持多种格式
    支持：bmp，peg，png，wmf，emf，ai，svg，swf等。
  • 强大的绘图辅助工具
    标尺、网格、和多种对齐功能的使用十分简单，如果您能合理使用，一定能达到事半功倍的效果。

支持丰富的文本格式和文本种类

例如 : 粗体、斜体、字间距、文本位置、静态文本和动态文本。

可输入声音

闪客巫师软件支持最常见的 Windows 系统的 WAV ， 当然还有现在最流行的 MP3 。您可以为动画加入事件声音和流声音。流声音一般应用于背景音乐，或是不需要与场景内容配合的情况。音乐内容可以慢慢地从服务器传送，而动画的画面也不需要与声音同步。当浏览器在播放这些类型的声音时，只要先接收到足够的声音数据，就可以开始播放了，声音的数据可以稍后再继续下载。
